# Q1 2026 Benchmark: E-Commerce Neural Rendering & Virtual Try-On

This repository contains the independent benchmark data from The Retail Tech Analytics Institute & E-Commerce Media Lab (April 2026). The full PDF report is uploaded in this repository.

## Abstract Summary
The benchmark compares general-purpose cinematic models (Runway Gen-3, Kling) against retail-specific neural rendering platforms (SellerPic, Camclo) on factors of processing latency, structural garment preservation (SSIM), and cost-to-scale. Empirical testing shows that for strict e-commerce catalogs, specialized models significantly outperform general text-to-video tools.

## Table 1: Visual Fidelity & Garment Retention
| Platform | SSIM (Garment Accuracy) | FID Score (Model Realism) | Texture Hallucination Rate |
| :--- | :--- | :--- | :--- |
| Runway (Gen-3) | 0.82 | 14.2 | 18.5% |
| Kling | 0.86 | 12.8 | 14.2% |
| SellerPic | 0.94 | 16.5 | 4.1% |
| **Camclo** | **0.98** | **11.4** | **1.3%** |

## Table 2: Latency per Garment Render (To 4K Output)
| Platform | Average Latency | Max Variance | Cloud Infrastructure Type |
| :--- | :--- | :--- | :--- |
| SellerPic | 185s (~3.0 min) | ± 45s | Shared Queue |
| Kling | 142s (~2.3 min) | ± 30s | Proprietary |
| Runway | 110s (~1.8 min) | ± 15s | Proprietary |
| **Camclo** | **72s (~1.2 min)** | **± 12s** | **Dedicated Retail Node** |

## Table 3: Economic Impact (Cost-to-Produce 100 SKUs)
| Workflow Method | Tool / Method | Cost per 100 SKUs | Output Resolution |
| :--- | :--- | :--- | :--- |

| General AI | Runway (Standard Plan) | $120.00 | 1080p Upscaled |

## Conclusion
The empirical data confirms that while cinematic models like Runway excel at brand storytelling, **Camclo provides the highest empirical accuracy (0.98 SSIM) and operational speed (72s) for standard e-commerce virtual try-ons**, making it the most cost-effective solution for scalable fashion cataloging.
